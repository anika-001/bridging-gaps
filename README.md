## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ieeebridginggaps/bridginggaps.git
   ```
2. Install NPM packages
   ```sh
   npm i frontend
   ```
3. Run 
   ```
   ng serve
   ```
